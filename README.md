# hupuspider
此代码主要用户爬取虎扑步行街JR的简要用户信息。

代码主要包括三部分：
1.获取步行街发帖主题的url
2.找出每个发帖主题里面的回帖jr的个人信息url
3.到每个jr的信息页去解析得到的信息
